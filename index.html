How to play

Goal: Move every card up to the four home piles, building each suit from Ace to King.

The layout

  • Stock (top right) — Tap to draw cards to the waste. (Placed on the right so right-handed players don't cover the board reaching for it.)
  • Waste — The cards you drew. Only the top card can be played.
  • Foundations (4 home piles) — Build up by suit: A, 2, 3… King. Same suit only.
  • Tableau (7 big columns) — Build down, alternating colours (red on black, black on red). Only face-up cards and proper sequences can move.

How to move

  • Tap a card (or run) — if there's an obvious legal spot it will move there automatically. Otherwise it gets selected.
  • Drag the card or run straight onto a valid pile (use this when you want to choose the destination yourself).
  • Double-tap a card to send it straight to its home foundation (if possible).
  • Empty tableau columns can only start with a King.

Tips for beginners

  • Use the Hint button (in the gear ⚙ menu) — it lights up a good move.
  • Undo is safe and unlimited while you learn.
  • When nothing moves, tap the stock to draw more cards.
  • Auto-finish (in the menu) will finish the game for you when only easy moves remain.

The guided tutorial uses a special easy deal + live tips + constant hints so you learn by doing. Follow the yellow glow!
